TWO MODES
---------

1. Hybrid Drift Field (Default)
   - For: Differentiation, development, trajectory inference
   - Features: Score + residual + velocity prior
   - Notebook: paul15_hybrid_drift_analysis.ipynb

2. Schrödinger Bridge (Optional)
   - For: Aging, perturbation, reprogramming
   - Features: Optimal transport, bidirectional
   - Notebook: schrodinger_bridge_synthetic_analysis.ipynb

QUICK START
-----------

1. Extract archive:
   tar -xzf scidiff_final.tar.gz
   cd scidiff_final

2. Install:
   python3.11 -m venv scidiff_env
   source scidiff_env/bin/activate
   pip install torch numpy scipy matplotlib tqdm scikit-learn \
               anndata scanpy scvelo pandas h5py numba pytest scikit-misc
   pip install -e .

3. Test:
   python test_final_repository.py

4. Explore:
   cd examples/
   jupyter notebook
