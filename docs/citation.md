# Citation

If you use scJDO in scientific work, please cite the project and any datasets or upstream software packages used in your analysis.

> Redd D., Green S., Terooatea T. W. (2026). *scJDO: Inferring time-varying dynamical operators from single-cell transcriptomic data.* [journal].

## Software citation checklist

| Item | Why it matters |
|---|---|
| scJDO version | Ensures the API and model implementation are identifiable |
| Dataset accession or source | Makes the biological input reproducible |
| Preprocessing parameters | Pseudotime and representation choices affect dynamics |
| Training parameters | Drift and bridge results depend on optimization settings |
| Random seed | Supports reproducible model fitting and archetype extraction |
